# Lab1_IT

Лабораторна робота №1-2 з Git.

##  Опис
Цей проєкт — інтерактивна консольна програма на Python, яка демонструє:
- арифметичні операції (додавання, віднімання, множення, ділення)
- перевірку паліндромів
- обчислення факторіалу
- управління списком завдань (ToDo: додавання, видалення, позначення виконаних)
- збереження та завантаження даних у JSON‑файл
- інтерактивне меню для вибору дій користувачем

## Приклад роботи
При запуску програма показує меню:
   1. Greeting
   2. Math operations
   3. Palindrome check
   4. Factorial
   5. Manage ToDo list
   6. Save/Load data
   0. Exit

Користувач вводить номер дії, після чого виконується відповідна функція.
Після кожної операції потрібно натиснути пробіл + Enter, щоб повернутися до меню.
