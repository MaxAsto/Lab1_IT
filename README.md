# Lab1_IT

Лабораторна робота №1 з Git.

## Ідея MVP
Простий Python-проєкт, який демонструє базову структуру репозиторію:
- вихідний код у папці `src/`
- тести у папці `tests/`
- `.gitignore` для ігнорування зайвих файлів
- README.md для опису проєкту

## Вимоги
- Python 3.10+
- Git
